Nieuw hoofdstuk

Sinds ik met programmeren ben begonnen is er een hele nieuwe wereld voor me geopend. Wat ik ondervonden heb is dat je enorm veel inzicht nodig hebt voor dit beroep. ook geduld is zeker een pluspunt als je wilt leren programmeren. Mijn droom is om later na mijn studie games te kunnen ontwikkelen. Ik game al erg lang en heb het altijd gewild om zelf een game op de markt te kunnen brengen. het concept heb ik al lang in mijn hoofd enkel de vaardigheden voor het spel te kunnen ontwikkelen heb ik nog nodig. als student programmeur hoop ik dan ook als ik in de periode kom dat de stages beginnen dat ik in zo'n omgeving mijn stage kan lopen. op stages kan je heel veel bijleren. Wat ik ook heel leuk vind aan programmeren is dat je heel creatief moet zijn en dat je heel vrij bent in je creaties. je kan echt letterlijk schrijven wat er maar in je op komt,dit wil natuurlijk niet zeggen dat het altijd gaat werken. Deze website is mijn eerste website die ik geschreven heb in Opdracht van mijn docent. Ik heb mijn uiterste best gedaan voor deze zo mooi mogenlijk af te werken.
